# Feature-Extraction-Assignment
MatLab project that implements a feature extraction technique on a given image (source image included in project)

This program was an assignment written for CSCI 158: 'Intro to Biometric Security' @ Fresno State. It uses a rudamentary image containing seperated shapes to be identified by the programming (used-image doesn't require preprocessing techniques). The program classifies each distinct shape by coloring the output image's shapes a distinct, randomly-generated RGB value. Performs feature extraction via local connectivity: determining if two neighboring pixels are apart of the same shape by testing their intensity values against eachother.
